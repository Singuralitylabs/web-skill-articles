# web-skill-articles

Web系技術の記事を管理するリポジトリです。記事はWebサービス「note」で公開予定です。

## 対象トピック

- GAS（Google Apps Script）の文法・アプリ紹介
- 生成AIを使った開発
- HTML/CSS
- React / Next.js
- その他Web技術全般

## ディレクトリ構成

```
.
├── ai_development/    # AI駆動開発に関する記事
│   ├── 01_ai-tools.md       # AI駆動開発ツール徹底比較 2025年版
│   ├── 02_context.md        # コンテキストエンジニアリングを学ぼう
│   └── 03_prompt.md         # AI駆動開発におけるプロンプトの書き方
├── columns/           # コラム記事
│   ├── 01_GASに拘る訳.md     # なぜ私たちはGAS受託開発にこだわるのか
│   └── 02_AI駆動開発の衝撃.md # AI駆動開発の衝撃
├── images/            # 記事で使用する画像素材
├── work/              # 作業用ファイル（HTML/CSS/JSなど）
├── CLAUDE.md          # Claude Code用の記事作成ガイドライン
└── README.md
```

## 記事のフォーマット

記事はMarkdown形式で作成し、noteのマークダウン記法に準拠しています。詳細なルールは `CLAUDE.md` を参照してください。
